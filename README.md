# FIFO Buffer in Verilog

FIFO (First-In First-Out) Buffer implemented in Verilog and simulated using Vivado 2021.2.

## Features

* FIFO memory buffer
* Write pointer control
* Read pointer control
* Full flag generation
* Empty flag generation
* Behavioral simulation
* Testbench verification

## Tools

* Verilog
* Vivado 2021.2
* Arty A7-100 FPGA board

* fifo.v
* fifo_tb.v
